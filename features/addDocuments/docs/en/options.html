Create a Library to Manage Start Menu

To manage start menu by a library ...
  1. Focus on "Libraries", new a library named as "Start Menu"
  2. Right "Context Menu > Properties"

  3. "Add...:"

    %APPDATA%\Microsoft\Windows\Start Menu\Programs

  4. "Add...:"

    C:\ProgramData\Microsoft\Windows\Start Menu\Programs

    Click "Save public save location". Require administrator to modify this directory.

  5. Change the icon if you prefer. To add items:

    1. Drop new shortcuts into the library
    2. Make the library a send target shell:sendto

Enable Recovery Mode

How to do?
  1. Execute diskpart, select recovery partition.

  2. Execute:

    format quick fs=ntfs label=Recovery
set id=de94bba4-06d1-4d40-a16a-bfd50179d6ac
gpt attributes=0x8000000000000001
assign letter r
  3. Use 7zip to extract \sources\install.wim 1\windows\system32\Recovery into R:\

  4. Execute:

    reagentc /setreimage /path r:\recovery
reagentc /enable
mountvol r: /d

Grey Color Invert

Suitable for invert white color mode only apps, press Win + Ctrl + C to take effect. 

reg add HKCU\Software\Microsoft\ColorFiltering /v HotkeyEnabled /t REG_DWORD /f /d 1
reg add HKCU\Software\Microsoft\ColorFiltering /v FilterType /t REG_DWORD /d 2 /f
