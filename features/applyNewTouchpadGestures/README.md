# Prefer Touchpad Gesture

## English Instruction

- Three finger swipe left and right:
  lower/raise volume
- Three fingers swipe up and down:
  maximize/minimize the window
- Four-finger swipe down:
  close the window
- Four-finger swipe up:
  task view
- Four fingers swipe left and right:
  switch desktop

## 中文说明

- 三指左右滑动：降低/提高音量
- 三指上下滑动：最大/最小化窗口
- 四指向下滑动：关闭窗口
- 四指向上滑动：任务视图
- 四指左右滑动：切换桌面
