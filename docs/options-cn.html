使用“库”

易于管理开始菜单 ...
  1. 打开 “库”，新建名为 “开始菜单” 的库
  2. 右键菜单 “属性” 编辑

  3. “添加”：

    %APPDATA%\Microsoft\Windows\Start Menu\Programs

  4. “添加”：

    C:\ProgramData\Microsoft\Windows\Start Menu\Programs

    点击 “设置公共保存位置”。( 修改该目录需要权限 )

  5. 按需要“更改库图标”。添加项目到开始菜单：

    1. 把需要的快捷方式拖入库即可
    2. 使库成为发送目标 shell:sendto
